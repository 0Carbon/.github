# 0Carbon.in 🌍

> "Strategic Sustainability for a Net Zero World."

Welcome to the official repository for **0Carbon.in**. We are dedicated to building high-fidelity tools for carbon footprint estimation, energy management, and sustainable technical infrastructure.

## 🛠 What we do
We bridge the gap between **industrial energy expertise** and **modern software engineering**. Our current focus is the `zero-carbon-calculator`, a tool designed to provide transparent, data-driven insights into carbon emissions.

### Core Tech Stack
*   **Backend:** Python (Fast API)
*   **Frontend:** React
*   **Data & AI:** RAG (Retrieval-Augmented Generation), Vector DBs, and local LLM integration

## 🚀 Projects
*   **zero-carbon-calculator:** A robust engine for calculating industrial and personal carbon offsets.
